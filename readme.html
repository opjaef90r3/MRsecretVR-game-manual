本作品は、基本内容を同じとしながらも以下の2つのバージョンに分かれています

  • PCVR版
    • PCとVRヘッドセットを接続して使用
    • SteamVRを起動し、「MRsecretVR.exe」を実行してプレイ
  • Quest単体版
    • Meta Questシリーズ単体で動作
    • Quest本体に「MRsecretVR.apk」をインストールしてプレイ

以下に詳しい手順を記載します。

PCVR版のセットアップ手順

1. SteamVRのインストール

  1. PCでSteamを起動してください
  2. 画面上部の「ストア」から「SteamVR」を検索
    もしくはこちらから直接アクセス
  3. 「無料でインストール」を選択し、インストールを完了させてください

2. ゲームファイルの配置

  1. ダウンロードしたゲームファイルを展開してください
  2. 展開したフォルダ「MRsecretVR_PCVR」を以下のような条件の場所に配置してください

    日本語（全角文字）が含まれないフォルダに置く

    OK → C:\Games\MRsecretVR_PCVR
    NG → C:\ゲーム\MRsecretVR_PCVR

※全角文字が含まれるとゲーム内での入力が効かなくなる場合があります。

3. VRヘッドセットの接続とSteamVRの起動

PCVR専用機の場合

対応機種：Pimax、VIVE等
  1. VRヘッドセットをPCに接続
  2. SteamVRを起動

スタンドアロン機の場合

対応機種：Quest 2/Pro/3/3S、PICO 4等

PCVRを遊ぶには以下のいずれかの方法が必要です

1. Quest Link（Questのみ）
  1. Meta Quest Linkアプリをインストール
  2. Quest本体の設定→システム→リンク→オン
  3. Air Linkの場合はペアリングが必要です
  4. 接続後、Quest Linkのホーム画面が起動します
  5. ホーム画面からSteam VRを選択して起動します
2. Virtual Desktop（無線・別途購入必要）
  1. Quest本体にVirtual Desktopアプリをインストール
  2. PCにVirtual Desktopストリーマーをインストール
  3. Quest本体からVirtual Desktopを起動し、PCと接続
  4. 接続が成功するとQuest内にPC画面が表示されます
  5. 左メニューから「Launch SteamVR」を選択
3. Steam Link（Questのみ）
▶ Steam Link設定の解説動画を見る

各接続方法が正常に完了すると、SteamVRのホーム画面がヘッドセットに表示されます。その状態で次に進んでください。

4. ゲームの起動

  1. 配置した「MRsecretVR_PCVR」フォルダを開く
  2. フォルダ内の「MRsecretVR.exe」をダブルクリック
  3. ゲームが起動し、VRヘッドセットにゲーム画面が表示されます

これで準備は完了です。お楽しみください！

Quest単体版のセットアップ手順

現在、インストール方法は大きく2通りあります：

  • A 「SideQuest」を使用したインストール方法（PCが必要）
  • B Questアプリ「Mobile VR Station」を使用したインストール方法（Quest単体で可能）

A. 「SideQuest」を使用したインストール方法

1. 開発者モードの有効化

Meta Quest for Developersでの組織登録
  1. Meta Quest for Developersにアクセス
  2. Meta Questのアカウントでログイン
  3. 本人確認のため以下のいずれかを設定
    • クレジットカード情報の登録（請求は発生しません）
    • または二段階認証の設定
  4. 組織名を登録

    非公式アプリをプレイするだけの場合は任意の名前で問題ありません

  5. 「理解しました」にチェックを入れて「送信」
  6. 開発者機密保持契約の「同意する」にチェックを入れて「送信」
スマホのMeta Questアプリでの開発者モード有効化
  1. スマホのBluetooth機能をオンにする
  2. スマホのMeta Questアプリを起動
  3. 三本線アイコンをタップしてメニューを開く
  4. 「デバイス」を選択
  5. お使いのQuestを選択（「接続済み」と表示されていることを確認）
  6. 「ヘッドセットの設定」をタップ
  7. 「開発者モード」をタップしてオンにする
  8. Quest本体を再起動

2. ADBドライバーのインストール

  1. Meta Quest開発者センターにアクセス
  2. 「Download」をクリック
  3. 「I have read and agree to the terms of the license .」にチェックを入れる
  4. 「Download」をクリックしてファイルを保存
  5. ダウンロードしたzipファイルを解凍
  6. 解凍したフォルダ内の「android_winusb.inf」を右クリックし「インストール」を選択
  7. 「この操作を正しく終了しました」と表示されたら「OK」をクリック

3. SideQuestのインストール

  1. SideQuest公式サイトにアクセス
  2. 「Advanced Installer」から「DOWNLOAD FOR WINDOWS」をクリック
  3. ダウンロードしたファイルを実行してインストール

4. QuestとPCの接続

  1. QuestとPCをUSBケーブルで接続
  2. Quest内で「USBデバッグを許可しますか？」→「許可する」を選択
  3. Quest内のホームメニュー下部のベルアイコンの「お知らせ」を選択
  4. 「USBが検出されました」を選択
  5. 「接続したデバイスにファイルへのアクセスを許可しますか？」→「許可する」を選択

5. ゲームのインストール

  1. SideQuestを起動
  2. SideQuest画面左上のマークが緑色になっていることを確認
    • Questの機種名が表示されていれば接続成功です
    • オレンジ色の場合は接続できていません。PCとの接続を確認してください
  3. 画面右上の「Install APK from folder」アイコン（フォルダマーク）をクリック
  4. ダウンロードした「MRsecretVR.apk」を選択
  5. インストール進行状況が画面左に表示されます
  6. インストール完了時に「MRsecretVR.apk: APK installed ok!!」と表示されます

6. ゲームの起動

  1. Questでアプリメニューを開く
  2. 左側メニューから
    • ハテナマークのフォルダアイコンを選択
    • 「提供元不明」を開く
  3. 「MRsecretVR」を選択して起動

※提供元不明が表示されない場合はQuest本体を再起動してください。

B. 「Mobile VR Station」を使用する方法

1. 開発者モードの有効化

前述のA方式と同様の手順で開発者モードを有効化してください

2. APKファイルの準備

APKファイルをQuest本体に用意する方法は2通りあります：

方法1: Quest単体で準備する場合
  1. QuestのブラウザからDLSiteにアクセス
  2. アカウントにログインしてゲームをダウンロード
  3. ダウンロードしたzipファイルをMobile VR Stationで解凍
    • Mobile VR Stationを開く
    • 「ローカルファイル - Local Files」を選択
    • 「ダウンロード - Downloads」を選択
    • ダウンロードしたゲームのzipファイルを選択
    • 「ファイルの抽出 - Extract Files」を選択
    • 「お待ちください」メッセージが表示→「OK」を選択
    • ダウンロードフォルダに解凍されたことを確認

※この時点ではapkファイルは見えません

方法2: PCから転送する場合
  1. PCでダウンロードしたゲームファイルを展開
  2. QuestとPCをUSB接続
  3. Quest内で「ファイルアクセスを許可」を選択
  4. PCからQuestの「download」フォルダに「MRsecretVR.apk」をコピー

3. Mobile VR Stationのインストール

Meta Quest ストアからインストールしてください。

4. APKのインストール

  1. Mobile VR Stationを起動
  2. 「構成ウィザード - Configuration Wizard」→「Show All Options」→「Configure Scoped Storage」→再度「Configure Scoped Storage」を選択
  3. 「Step1: Request Access」を選択
  4. 「Quest」→「Download」フォルダを開く
  5. 「MRsecretVR_ver**」フォルダ内の「MRsecretVR.apk」を確認
  6. APKファイルの右上の矢印アイコンを選択
  7. 確認画面で「次へ」→「インストール」を選択
  8. インストール完了したら「完了」を選択
  9. Mobile VR Stationを終了

5. ゲームの起動

  1. Questでアプリメニューを開く
  2. 左側メニューから
    • ハテナマークのフォルダアイコンを選択
    • 「提供元不明」を開く
  3. 「MRsecretVR」を選択して起動

※提供元不明が表示されない場合はQuest本体を再起動してください。

これで準備は完了です。お楽しみください！
