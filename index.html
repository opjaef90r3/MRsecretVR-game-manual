対応機種

PCVR版

  • 対応OS: Windows 10/11
  • 必須: SteamVR
  • 推奨GPU: RTX3060以上
  • MR: クロマキーによるパススルーに対応

Quest単体版

注意事項
  • SideQuestによるインストールが必要です。
対応機種
  • Meta Quest 2
  • Meta Quest Pro
  • Meta Quest 3
  • Meta Quest 3S
機能制限
  • Meta Quest 2：一部機能制限あり（販売ページをご覧ください）
  • Meta Quest Pro：一部機能制限あり（販売ページをご覧ください）
  • Meta Quest 3：完全対応・空間認識機能対応
  • Meta Quest 3S：完全対応・空間認識機能対応
