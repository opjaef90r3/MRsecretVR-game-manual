インストール方法

PCVRバージョン

  1. SteamVRをインストールしてください
  2. ゲームフォルダを全角文字の含まれないパスに移動してください
  3. SteamVRを起動し、VRヘッドセットとの接続を確認してください
  4. stealthHeart.exeを実行してください

Quest単体バージョン

  1. SideQuestをインストールしてください (Advanced Installerを推奨)
  2. QuestをPCに接続してください
  3. SideQuestでMRsecretVR.apkをインストールしてください
  4. Questで「提供源不明のアプリ」からゲームを起動してください

注意事項

  • 本作品は18歳未満の方はプレイできません
  • VR機器の安全ガイドラインに従ってプレイしてください
  • 定期的に休憩を取ることを推奨します
  • プレイ中に気分が悪くなった場合は直ちに中止してください

動作環境

PCVRバージョン

  • 対応OS: Windows 10, 11
  • 必須: SteamVR
  • 推奨GPU: RTX3060以上
  • 主要VRヘッドセットに対応

Quest単体バージョン

  • Meta Quest 2 (モノクロパススルー、空間認識機能は非対応)
  • Meta Quest 3/3S
  • Meta Quest Pro (空間認識機能は非対応)
